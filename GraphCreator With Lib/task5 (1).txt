Написать класс сигнала Signal, хранящий массив значений дискретного сигнала и его частоту дискретизации.
Класс должен самостоятельно обслуживать память массива значений.
Класс должен реализовывать следующие методы:

- конструктор по-умолчанию
- конструктор, принимающий размер типа size_t
- метод resize, изменяющий размер массива
- перегрузить операторы []
- перегрузить оператор + для сложения двух сигналов
- перегрузить оператор - для вычитания двух сигналов
- перегрузить операторы += и -=
- поддерживать двоичную сериализацию (в поток/из потока)

Переписать лабораторную работу по определению амлитуды сигнала таким образом, чтобы она оперировала 
объектами класса Signal, а не массивами.

Написать программу с реализацией своего класса на основе QWidget, реализующего отображение сигнала 
в виде графика зависимости значений от времени. 
Класс должен отображать на графике шкалы времени и величины сигнала. Шкала величины должна располагаться 
с левой стороны. Шкала времени в месте расположения нулевого значения величины, либо в нижней или верхней
части виджета, в зависимости от того, где располагается нулевое значение.
Класс должен масштабировать сигнал по времени и по величине с зафиксированной
точкой, в которой находится курсор мыши. Масштабирование по величине осуществляется колёсиком мыши.
Масштабирование по времени колёсиком с зажатым Shift.
